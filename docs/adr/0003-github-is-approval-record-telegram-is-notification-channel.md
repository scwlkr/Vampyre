# Use GitHub for approval records and Telegram for notifications

GitHub will be Vampyre's formal approval and review surface, while Telegram will be used for notifications, blocker alerts, status pings, and links to GitHub or visual reports. This keeps durable decisions near issues, PRs, labels, and comments while still giving the Owner quick interrupt-style updates away from the repo.

## Consequences

- Significant approvals must be captured in GitHub through issues, PRs, comments, or labels.
- Telegram-only approval is not enough for significant work.
- Missing Telegram configuration can block MVP completion, but should not stop all GitHub-centered daemon work.
